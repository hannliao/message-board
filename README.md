# message-board

Message board using NodeJS/Express and Model-View-Controller (MVC) pattern.

## Features

- View all messages
- Post a new message
- Open any message to view

## Built with

- [NodeJS](https://nodejs.org/en)
- [Express](https://expressjs.com/)
- JavaScript
- [EJS](https://ejs.co/)
- CSS
- [uuidv4](https://www.npmjs.com/package/uuidv4)
